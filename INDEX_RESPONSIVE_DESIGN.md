## üéØ Quick Answer Guide

### Q: What device sizes should I support?
A: See `RESPONSIVE_QUICK_REFERENCE.md#cheat-sheet` or `README_RESPONSIVE_DESIGN.md#device-categories`

### Q: What padding should I use?
A: Mobile 12, Tablet 16, Desktop 24 - use `responsivePadding(context)`

### Q: What font sizes should I use?
A: Use `headlineSize()`, `titleSize()`, `bodySize()`, `smallSize()` - see `RESPONSIVE_QUICK_REFERENCE.md#font-sizes`

### Q: How big should my buttons be?
A: Min 44√ó44 dp on mobile - see `RESPONSIVE_QUICK_REFERENCE.md#component-sizes`

### Q: How do I detect device type?
A: Use `isMobile()`, `isTablet()`, `isDesktop()` - see `RESPONSIVE_QUICK_REFERENCE.md#device-detection`

### Q: How tall should my map be?
A: 35% on mobile, 45% on tablet, 55% on desktop - use `mapHeight()`

### Q: Where's the code I need to copy?
A: `RESPONSIVE_IMPLEMENTATION_EXAMPLE.md` has all the copy-paste examples

### Q: I need help visualizing layouts
A: Check `RESPONSIVE_DESIGN_VISUAL.md` for ASCII art examples

### Q: Which document should I read first?
A: `RESPONSIVE_QUICK_REFERENCE.md` (5 min quick start)

### Q: I'm stuck on a problem
A: Check `README_RESPONSIVE_DESIGN.md#troubleshooting` or `RESPONSIVE_DESIGN_GUIDE.md#troubleshooting`
